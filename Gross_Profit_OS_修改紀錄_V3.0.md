# Gross Profit OS V3.0 - 修改紀錄

## 版本：3.0 Enterprise
## 修改日期：2026-03-08
## 基於：V2.0 Ultimate

---

## 新增頁面（4個）

| 頁面名稱 | 功能說明 |
|---------|---------|
| 🚨 預警中心 | 即時監控毛利異常、連續下跌偵測、低毛利產品警示、側邊欄紅色數字徽章 |
| 🌊 毛利瀑布分解 | 量效應/價效應/成本效應三維分解、月份選擇器、瀑布圖視覺化 |
| 🎯 最佳定價分析 | 售價×銷量×毛利率三維散點圖、毛利率vs毛利額組合圖、峰值點識別 |

## 新增功能

### 1. 供應商分析頁面（全新）
- 供應商採購金額佔比甜甜圈圖
- 加權平均採購成本對比（含正確/錯誤做法說明）
- 供應商集中度風險指標
- 供應商明細表（含採購佔比進度條）

### 2. 資料結構擴充
- 新增「供應商數據」工作表（6個欄位）
- 新增「目標設定」工作表（指標名稱/目標值/警戒值/單位）
- 月度數據新增「去年同期成本」和「目標收入」欄位
- 成本數據新增「去年金額」欄位（支援 YoY 對比）

### 3. JSON 匯入功能（新增）
- 支援直接匯入 JSON 格式資料
- 匯入後自動重新渲染所有圖表
- 支援 .xlsx / .xls / .csv / .json 四種格式

### 4. 匯出功能增強
- 新增「匯出 Excel」功能（含所有9個工作表）
- 保留原有 JSON 匯出功能
- 新增「列印/PDF」按鈕（使用瀏覽器列印功能）

### 5. DAX 知識庫擴充（12條→16條）
新增公式：
- ⑬ 毛利瀑布分解（量效應/價效應/成本效應）
- ⑭ 動態標題 SELECTEDVALUE
- ⑮ 動態警戒線 Parameter + SWITCH
- ⑯ 星型結構數據模型架構說明

知識庫改為分頁式（基礎指標/時間智能/進階分析/視覺化），可折疊展開。

### 6. 預警中心功能
- 側邊欄即時顯示預警數量（紅色徽章）
- 自動偵測：整體毛利率低於警戒線、連續下跌月數、低毛利產品數、成本增幅超標
- 連續下跌偵測圖表（對應 DAX 公式）
- 低毛利產品清單（含狀態標籤）

### 7. 瀑布圖分解
- 月份選擇器（可選擇任意月份進行分析）
- 瀑布圖使用堆疊柱狀圖模擬（純 Chart.js 實現）
- 量效應/價效應/成本效應分別顯示
- 正值=綠色，負值=紅色

### 8. 熱力圖
- 地區×月份矩陣（5地區×12月份）
- 顏色漸層：紅色（低）→ 綠色（高）
- hover 顯示精確數值
- 配套地區平均柱狀圖和趨勢折線圖

## 技術改進

- 新增 `--teal` 和 `--pink` CSS 變數
- 新增 `.nav-badge` 側邊欄徽章樣式
- 新增 `.nav-section` 導航分組標題
- 新增 `.progress-bar` 進度條元件
- 新增 `.tabs` 分頁切換元件
- 新增 `.heatmap-cell` 熱力圖格子樣式
- `renderAll()` 函數統一重新渲染當前頁面
- `updateAlertBadge()` 函數即時更新預警數量
- 支援 `@media print` 列印樣式

## 資料範例更新

- 月度數據：新增去年同期成本和目標收入欄位
- 成本數據：新增去年金額欄位
- 供應商數據：6家供應商，含採購金額/數量/單位成本/品類/付款條件
- 目標設定：5項指標目標值和警戒值

---

**V3.0 核心設計理念**：「從分析到行動」
- 瀑布圖讓管理者一眼看出毛利變動的根因
- 預警中心讓問題在惡化前被發現
- 最佳定價分析直接支援定價決策
- 供應商分析幫助優化採購成本
