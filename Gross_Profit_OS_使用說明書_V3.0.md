# Gross Profit OS V3.0 使用說明書

## 版本：3.0 Enterprise
## 適用對象：財務分析師、業務主管、管理決策者

---

## 一、快速開始

### 1.1 開啟方式
直接用瀏覽器開啟 `Gross_Profit_OS_V3.0.html`，無需安裝任何軟體。

建議瀏覽器：Chrome / Edge（最佳相容性）

### 1.2 使用範例資料
首次開啟時自動載入範例資料（服飾/飾品/鞋類零售業）。
點擊左側「↺ 重置範例資料」可隨時恢復預設值。

---

## 二、資料匯入

### 2.1 支援格式
| 格式 | 說明 |
|------|------|
| `.xlsx` / `.xls` | Excel 工作簿（推薦） |
| `.csv` | 逗號分隔值 |
| `.json` | JSON 格式（從本系統匯出的資料） |

### 2.2 匯入方式
**方式一：點擊按鈕**
點擊頂部「📂 匯入數據」按鈕，選擇檔案。

**方式二：拖放**
將檔案直接拖放到瀏覽器視窗中。

### 2.3 Excel 工作表規格

| 工作表名稱 | 必要欄位 |
|-----------|---------|
| 月度數據 | 時間, 銷售收入, 銷售成本, 去年同期收入, 去年同期成本, 目標收入 |
| 產品數據 | 產品名稱, 類別, 銷售單價, 單位成本, 銷售總量, 相對市占率, 市場成長率, BCG象限 |
| 成本數據 | 成本大類, 成本小類, 金額, 去年金額 |
| 客戶數據 | 客戶名稱, LTV總額, 毛利貢獻率, 購買頻率, 客戶等級 |
| 地區數據 | 地區名稱, 毛利率, 配送效率, 售價溢價, 市場份額, 品牌知名度 |
| 渠道數據 | 渠道名稱, 銷售額, 成本, 渠道類型 |
| 促銷數據 | 活動名稱, 折扣率, 銷量增幅, 毛利變動, 活動類型 |
| 供應商數據 | 供應商名稱, 採購金額, 採購數量, 單位成本, 品類, 付款條件 |
| 目標設定 | 指標名稱, 目標值, 警戒值, 單位 |

> 💡 點擊「⬇ 範例Excel」下載包含所有工作表的範例檔案，直接修改數值後匯入。

### 2.4 資料更新後
匯入成功後，系統自動重新計算所有 KPI 並更新圖表。
頂部顯示「最後更新」時間戳記。

## 三、功能頁面說明

### 3.1 📊 全局分析總覽
- **KPI 卡片**：年度收入、毛利、毛利率、目標達成率（含 YoY 趨勢）
- **月度毛利率趨勢**：含目標線（35%）和去年同期對比
- **產品毛利率排行**：顏色標示高/中/低毛利
- **收入 vs 成本**：月度柱狀圖
- **渠道毛利貢獻**：甜甜圈圖
- **成本結構**：直接/間接成本佔比

### 3.2 🚨 預警中心
- **側邊欄紅色數字**：即時顯示活躍預警數量
- **自動偵測項目**：
  - 整體毛利率低於警戒線（預設 30%）
  - 毛利率連續下跌月數 ≥ 3
  - 低毛利產品數量（毛利率 < 35%）
  - 成本增幅超過 10%
- **連續下跌偵測圖**：對應 DAX 公式「連續下降天數」
- **低毛利產品清單**：含狀態標籤（正常/注意/警告）

### 3.3 🌊 毛利瀑布分解
- **月份選擇器**：選擇任意月份進行分析
- **瀑布圖**：分解毛利變動為量效應、價效應、原材料、物流、其他
- **量效應 vs 價效應**：銷量變動和售價變動對毛利的影響
- **成本效應分解**：各成本項目變動的影響

### 3.4 ⚙️ 利潤情境模擬
- **四個調整滑桿**：售價、銷量、原材料成本、折扣率
- **即時計算**：調整後立即顯示模擬毛利率和毛利額
- **三情境對比圖**：悲觀/基準/模擬/樂觀四種情境
- **重置按鈕**：一鍵恢復所有參數為 0

### 3.5 📈 時間序列趨勢
- **警戒線設定**：可自訂警戒線百分比（預設 30%）
- **移動平均線**：3期移動平均（橙色虛線）
- **異常點標示**：低於警戒線的月份以紅色圓點標示
- **YoY 對比**：本期 vs 去年同期柱狀圖

### 3.6 📦 產品組合分析 (BCG)
- **BCG 矩陣**：氣泡大小=銷售額，顏色=毛利率
- **四象限**：明星（高市占/高成長）、金牛（高市占/低成長）、問題（低市占/高成長）、瘦狗（低市占/低成長）
- **產品毛利率排行**：橫向柱狀圖
- **產品明細表**：含 BCG 象限標籤

### 3.7 🎯 最佳定價分析
- **三維散點圖**：X軸=售價，Y軸=銷量，氣泡大小=毛利率
- **組合圖**：柱狀=毛利額，折線=毛利率（雙軸）
- **核心洞察**：找到毛利「額」的峰值點（非毛利率最高點）

### 3.8 🌍 地域渠道分析
- **地區雷達圖**：五維度（毛利率/配送效率/售價溢價/市場份額/品牌知名度）
- **渠道效率組合圖**：銷售額/毛利額柱狀 + 毛利率折線（雙軸）
- **地區績效總表**：含綜合評分

### 3.9 🔥 地區熱力圖
- **矩陣視圖**：5地區 × 12月份
- **顏色說明**：紅色=低毛利率，綠色=高毛利率
- **hover 提示**：顯示精確數值
- **配套圖表**：地區平均柱狀圖和各地區趨勢折線圖

### 3.10 💰 精細成本拆解
- **成本樹狀圖**：面積=金額佔比，直接/間接成本分組
- **帕累托分析**：識別佔 80% 成本的關鍵項目（20/80 法則）
- **YoY 成本對比**：本期 vs 去年同期

### 3.11 👥 客戶價值分析
- **LTV vs 毛利貢獻氣泡圖**：氣泡大小=購買頻率
- **客戶等級分佈**：鑽石/金/銀/銅
- **客戶明細表**：按 LTV 排序

### 3.12 🎯 折扣促銷評估
- **折扣率 vs 銷量增幅散點圖**：綠色=毛利正向，紅色=毛利負向
- **促銷 ROI 排行**：橫向柱狀圖
- **促銷明細表**：含評級（優秀/普通/虧損）

### 3.13 🏭 供應商分析
- **採購金額佔比**：甜甜圈圖
- **單位成本對比**：含加權平均線
- **集中度風險**：最大供應商佔比超過 40% 顯示警告
- **供應商明細表**：含採購佔比進度條

### 3.14 📖 DAX 知識庫
- **分頁式**：基礎指標/時間智能/進階分析/視覺化
- **16條完整公式**：含說明、代碼、常見陷阱
- **可折疊展開**：點擊標題展開/收起

## 五、常見問題

**Q：匯入後圖表沒有更新？**
A：確認 Excel 工作表名稱完全一致（包含中文），可點擊「❓ 匯入格式說明」確認。

**Q：預警中心顯示的數字是什麼？**
A：側邊欄紅色數字代表目前活躍的預警項目數量（不含「所有指標正常」）。

**Q：如何調整警戒線？**
A：在「時間序列趨勢」頁面右上角可輸入自訂警戒線百分比。

**Q：瀑布圖的量效應/價效應如何計算？**
A：量效應 = 銷量變動 × 去年售價 × 去年毛利率；價效應 = 本期銷量 × 售價變動 × 去年毛利率。

**Q：如何列印報告？**
A：點擊「🖨 列印/PDF」，在瀏覽器列印對話框中選擇「另存為 PDF」。
