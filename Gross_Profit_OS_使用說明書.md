# 毛利分析互動式看板 — 使用說明書

> **檔案名稱**：`Gross_Profit_OS.html` &nbsp;|&nbsp; **版本**：v1.0

---

## 一、快速開始

### 系統需求
- 任何現代瀏覽器：Chrome / Edge / Firefox（建議 Chrome 最新版）
- **不需要**安裝任何軟體或伺服器

### 開啟方式
```
直接雙擊 Gross_Profit_OS.html 即可在瀏覽器中開啟
```

> ⚠️ **注意**：Excel 匯入功能需要網路連線（用於載入 SheetJS CDN）。圖表在離線環境下也可正常瀏覽（初次載入後）。

---

## 二、介面說明

### 2.1 頁首工具列
```
📊 毛利分析 OS           [時鐘] [📂 匯入資料] [💾 匯出 JSON] [↺ 重置]
──────────────────────────────────────────────────────
💡 匯入狀態列（顯示目前使用的資料來源與匯入時間）
```

| 按鈕 | 功能 |
|-----|------|
| `📂 匯入資料` | 開啟檔案選擇對話框，匯入 Excel/CSV |
| `💾 匯出 JSON` | 將目前資料匯出為 JSON 備份檔 |
| `↺ 重置` | 恢復為內建範例資料 |

### 2.2 頁籤切換
點擊頂部任一頁籤即可切換分析視圖，所有頁籤均可同時顯示最新資料。

---

## 三、八大頁籤功能說明

### 📊 頁籤 0：總覽儀表板
**目的**：快速掌握企業健康狀態

**包含元素**：
- 🔴🟡🟢 **智慧警示**：毛利率達標狀態、連續下跌預警
- **4 個 KPI 卡片**：銷售收入、銷售成本、銷售毛利、毛利率（含 YoY 徽章）
- **毛利率目標達成儀表**：弧形量表 + 進度條，目標線為 35%
- **連續下跌月數**：紅色（≥3月）/ 黃色（1-2月）/ 綠色（0月）
- **月度趨勢折線圖**：收入 + 毛利雙線
- **Top 5 產品毛利率橫條圖**
- **各地區毛利率比較**

---

### 📈 頁籤 1：時間序列分析
**目的**：識別趨勢方向、異常波動、季節規律

**包含元素**：
- 分析期間、最低/最高毛利率月份、波動幅度 KPI
- **主趨勢圖**（切換按鈕）：
  - `收入`：月度銷售收入折線
  - `毛利率%`（預設）：毛利率 + 3 個月移動平均 + 警戒線
  - `毛利額`：月度毛利絕對值
- **月度堆疊圖**：收入/成本/毛利並列
- **下跌月份識別圖**：綠色=上漲月、紅色=下跌月

---

### 📦 頁籤 2：產品線 & BCG 矩陣
**目的**：識別明星產品、優化產品組合

**包含元素**：
- **產品毛利率橫條圖**（從高到低排序，顏色預警）
- **BCG 氣泡矩陣**：
  - X 軸：相對市場占有率
  - Y 軸：市場成長率
  - 氣泡大小：銷售額
  - 顏色：⭐明星（藍）/ 🐄金牛（綠）/ ❓問題（橘）/ 🐕瘦狗（紅）
- **毛利貢獻度甜甜圈圖**（= 收入 × 毛利率）
- **產品詳細數據表格**

---

### 🌍 頁籤 3：地域渠道分析
**目的**：找到高潛力地區與最佳渠道組合

**包含元素**：
- 最高/最低毛利率地區 KPI、渠道差異幅度
- **地區毛利率條形圖**（排序 + 顏色預警）
- **渠道 ROI 雙軸圖**：毛利率% + ROI 值比較
- **詳細數據表格**：含物流成本率、ROI、策略建議

---

### 💰 頁籤 4：成本結構分析
**目的**：識別關鍵成本項目，優化帕累托 20/80

**包含元素**：
- 原材料/人工/物流/固定成本比例 KPI
- **成本佔比圓餅圖**（10 項成本分類）
- **帕累托分析圖**（條形 + 累積佔比折線 + 80% 紅色虛線）
- **詳細成本表**：含優化優先級（P1/P2/P3）

> 💡 **解讀邏輯**：累積佔比達到 80% 以前的成本項目，是「P1 最高優先」優化目標。

---

### 👥 頁籤 5：客戶群體分析
**目的**：RFM 分層，聚焦高價值客戶

**包含元素**：
- VIP 客戶數、平均客單價、購買頻率、頂級客戶毛利貢獻 KPI
- **RFM 氣泡圖**：
  - X 軸：購買頻率（Frequency）
  - Y 軸：LTV（近似 Recency 代替）
  - 氣泡大小：客單價（AOV）
  - 顏色：VIP（藍）/ Premium（綠）/ Standard（橘）/ Basic（紅）
- **LTV 橫條排名圖**（前 10 名）
- **客戶詳細數據表**：含升級策略建議

---

### 🎯 頁籤 6：促銷折扣分析
**目的**：衡量促銷效益，防止品牌溢價侵蝕

**包含元素**：
- 平均折扣率、銷量增幅、促銷 ROI、品牌侵蝕風險 KPI
- **⚙️ 動態警戒線 SWITCH 參數**：
  - 四個滑桿分別設定服飾類/飾品類/鞋類/通用警戒線
  - 所有超過警戒線的促銷活動自動在表格中標紅
- **折扣率 vs 銷量增幅散點圖**：識別「高折扣低效益」的問題活動
- **促銷 ROI 橫條排行**
- **促銷活動詳細記錄表**

---

### 📖 頁籤 7：DAX 公式速查
**目的**：Power BI 開發時的公式參考手冊

**包含元素**（8 張可展開卡片）：
1. 銷售收入
2. 銷售毛利率（含 DIVIDE 陷阱說明）
3. 加權平均採購成本（含錯誤 vs 正確對比）
4. 零庫存成本修正（LASTNONBLANKVALUE）
5. 連續下跌天數預警（含矩陣 Grand Total Bug 修正）
6. SWITCH(TRUE) 多條件分類警戒線
7. CONCATENATEX 異常日期追蹤
8. SELECTEDVALUE 動態標題

**操作**：點擊任一卡片標題列即可展開/收合詳細說明。

## 四、資料匯入完整教學

### 4.1 準備 Excel 檔案

請準備一份 `.xlsx` 檔，包含以下工作表（**名稱需完全一致**）：

| 工作表名稱 | 必要欄位 |
|----------|---------|
| `月度數據` | month, revenue, cost, margin |
| `產品數據` | name, cat, revenue, margin, growthRate, share, bcg |
| `地區數據` | region, channel, margin, revenue, logisticRate, roi |
| `成本數據` | item, type, amount |
| `客戶數據` | name, region, tier, ltv, aov, freq, marginContrib |
| `促銷數據` | name, cat, discountRate, volumeInc, marginImpact, roi |

> 💡 **提示**：下載 `毛利分析範例資料.xlsx` 作為模板，直接修改資料即可，工作表名稱與欄位名稱**請勿變更**。

### 4.2 欄位格式說明

| 欄位 | 格式 | 範例 |
|-----|------|------|
| month | `YYYY-MM` | `2024-01` |
| revenue / cost / amount / ltv | 整數（元） | `3850000` |
| margin / growthRate / share / discountRate 等 | 小數（0-1） | `0.35` 代表 35% |
| bcg | 字串：`star` / `cashcow` / `question` / `dog` | `star` |
| tier | 字串：`VIP` / `Premium` / `Standard` / `Basic` | `VIP` |
| type（成本） | 字串：`固定` / `可變` | `固定` |
| channel | 字串：`直銷` / `電商` / `經銷` | `直銷` |

### 4.3 匯入步驟
1. 準備好 Excel 檔（可以只包含部分工作表，其餘保留範例資料）
2. 點擊「📂 匯入資料」**或**直接拖曳 Excel 檔到頁面任意位置
3. 狀態列顯示 **✅ 匯入成功** 後，所有圖表自動更新
4. 若顯示 **❌ 解析失敗**，請確認工作表名稱是否正確

### 4.4 支援 CSV 格式
若使用 `.csv` 格式，系統會嘗試解析為單一資料集（對應「月度數據」），其他維度數據需使用 Excel 格式。

## 五、常見問題

| 問題 | 解決方案 |
|-----|---------|
| 圖表顯示空白 | 確認瀏覽器版本支援 ES6+（Chrome/Edge 最新版） |
| Excel 匯入後圖表沒有更新 | 確認工作表名稱完全一致（中文名稱） |
| 拖曳無反應 | 嘗試點擊「📂 匯入資料」按鈕手動選擇檔案 |
| 字體顯示異常 | 確認網路連線以載入 Google Fonts |
| 警戒線滑桿沒有效果 | 切換至「🎯 促銷折扣」頁籤後再調整 |
