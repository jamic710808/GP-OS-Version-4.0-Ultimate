# 毛利分析 OS V2.0 使用說明書 (User Manual)

歡迎使用 **Gross Profit OS V2.0 Ultimate**。本版本為企業決策者量身打造，結合了 Power BI 的分析深度與網頁互動的極致體驗。

## 🚀 快速上手

### 1. 啟動儀表板
直接使用瀏覽器開啟 `Gross_Profit_OS_V2.0.html`。預設會載入內建的 2024 年範例數據。

### 2. 更新您的數據
您可以透過以下兩種方式更新儀表板內容：
*   **點擊按鈕**：點擊右上角「📂 匯入數據」選擇您的 Excel/CSV。
*   **直接拖放**：將檔案拖曳至瀏覽器視窗中任何位置。
*   **範例檔產出**：若您需要空白範例，請執行目錄下的 `generate_v2_data.py` 獲得標準格式的 XLSX 檔案。

## 🎮 特色功能解析

### ✅ 利潤情境模擬器 (Scenario Simulator)
位於側邊欄第二項。您可以針對以下維度進行「假設分析 (What-If Analysis)」：
1.  **調價模擬**：向右拉動單價，系統會根據價格彈性預測銷量下降後的總利潤。
2.  **銷量拉升**：模擬市場推廣成功後的獲利水平。
3.  **成本風險**：模擬原材料價格上漲時，對底線毛利的衝擊程度。

### ✅ 毛利瀑布圖 (Waterfall Chart)
在首頁總覽可見。它將「總營收」拆解為「變動成本」與「固定成本分攤」，最後結算出「實際毛利」，解決了長條圖無法呈現扣減過程的問題。

### ✅ BCG 矩陣組合分析
幫助您決定產品的未來命運：
*   **明星 (Stars)**：高成長、高占比 — **加大投資**。
*   **金牛 (Cash Cows)**：低成長、高占比 — **收割現金**。
*   **問題兒 (Question Marks)**：高成長、低占比 — **決定是否搶佔**。
*   **瘦狗 (Dogs)**：低成長、低占比 — **考慮淘汰**。

## 📊 數據匯入格式規範

匯入的 Excel 必須包含以下指定工作表名稱，否則引擎無法解析：

| 工作表名稱 | 核心欄位要求 |
| :--- | :--- |
| **月度數據** | `時間`, `銷售收入`, `銷售成本`, `去年同期收入` |
| **產品數據** | `產品名稱`, `類別`, `銷售單價`, `單位成本`, `銷售總量`, `市場成長率` |
| **成本數據** | `成本大類`, `成本小類`, `金額`, `屬性(固定/可變)` |
| **客戶數據** | `客戶名稱`, `LTV總額`, `毛利貢獻率` |

## 🛠️ 技術故障排除
*   **圖表沒出現**：請確保您的電腦已連上網際網路（需載入 Chart.js 庫）。
*   **匯入報錯**：請確保欄位名稱完全匹配上述規範，且無多餘的合併單元格。
*   **重置資料**：點擊側邊欄底部的「↺ 重置範例資料」即可恢復預設狀態。
