# 毛利分析 OS V2.0 修改紀錄

## 版本資訊
*   **版本號**：V2.0 Ultimate
*   **更新日期**：2026-03-05
*   **主要目標**：提升互動深度、新增決策模擬、優化視覺層次

## 重大改動彙整

### 1. UI/UX 視覺重構
*   **側邊欄導覽**：捨棄頂部 Tab，改用專業級左側收納式導航列（Sidebar），擴大 20% 圖表可視面積。
*   **精確擬玻璃 (Glassmorphism)**：使用 `backdrop-filter` 與精細的 `linear-gradient` 邊框，營造高端財務控制塔質感。
*   **動態過渡動畫**：頁面切換時加入 `slideUp` 動畫，提升操作流暢感。

### 2. 新增殺手級功能：利潤情境模擬器 (Scenario Simulator)
*   **動態滑桿控制**：允許即時調整「平均單價」、「銷售總量」與「單位成本」。
*   **價格彈性引擎**：內建價格彈性算法（E=1.5），自動計算調價後的銷量變化與淨毛利額。
*   **敏感度圖表**：即時預測模擬後的毛利額與基準值的對比分布。

### 3. 進階分析圖表升級
*   **毛利瀑布圖 (Waterfall)**：在總覽頁首創瀑布圖形式，直觀呈現營收如何經過成本扣除後，轉化為最終毛利。
*   **YoY 年度增長分析**：時間序列頁面整合「去年同期」數據對比線（虛線網格）。
*   **3D BCG 氣泡矩陣**：升級產品分析維度，將銷量、成長率、佔有率整合於動態氣泡大小。
*   **帕累托 80/20 V2**：強化成本分析，直觀標示出應優先優化的「顯著少數」成本項。

### 4. 資料引擎 V2.0
*   **Excel 全域支援**：優化 SheetJS 解析邏輯，對應 V2.0 多維度 Excel 範例模板（月度、產品、地域、成本、客戶、促銷）。
*   **範例腳本支援**：同步發布 `generate_v2_data.py`，一鍵生成完美匹配新版介面的測試數據。
*   **檔案拖拽匯入**：實現全畫面拖拽匯入 Excel/CSV，極速更新實時儀表板內容。
