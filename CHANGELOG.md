### 新增功能

#### 1. 完整範例資料集
- **月度數據**：12 個月營收/成本/去年同期（含去年成本）
- **產品數據**：7 項產品，涵蓋服飾/飾品/鞋類三大品類
- **成本數據**：8 項成本，分直接/間接兩大類
- **客戶數據**：8 位客戶，含 LTV、毛利貢獻率、購買頻率、客戶等級
- **地區數據**：5 個地區（台北/新北/台中/高雄/桃園），含 5 維指標
- **渠道數據**：5 個渠道（品牌官網/電商/直營/百貨/經銷），含銷售額與成本
- **促銷數據**：8 項促銷活動，含折扣率/銷量增幅/毛利變動/活動類型

#### 2. 客戶價值分析頁（完整實作）
- LTV vs 毛利貢獻率氣泡圖（氣泡大小=購買頻率）
- 客戶等級分佈甜甜圈圖
- 客戶價值明細表

#### 3. 折扣促銷評估頁（完整實作）
- 折扣率 vs 銷量增幅散點圖
- 促銷活動 ROI 橫條圖（含評級）
- 促銷活動明細表

#### 4. 成本結構樹狀圖（HTML Treemap 取代隱藏 Canvas）
- 使用 CSS Flexbox 實作矩形佔比圖
- 互動式 hover 效果，顯示金額與佔比

#### 5. 渠道效率組合圖
- 銷售額/毛利額柱狀圖 + 毛利率折線圖（雙軸）
- 地區績效總表

#### 6. 情境模擬器修復
- 模擬結果直接用數值計算，不再從 DOM 文字反解析
- 新增毛利率變動（pp）與毛利額變動顯示
- 基準 vs 預測三指標對比圖（營收/成本/毛利）

#### 7. 時間序列頁面增強
- 新增目標線（35%）標記
- 新增月度營收與成本走勢柱狀圖
- 新增連續毛利率下跌偵測面板

#### 8. 全局分析總覽增強
- KPI 趨勢指標改為動態 YoY 計算
- 新增產品毛利率一覽表
- 預警系統增加低毛利產品偵測

#### 9. DAX 知識庫擴充
根據《Power BI 毛利分析完整知識報告》補齊至 12 條完整 DAX 公式：
1. 銷售收入 SUM
2. 銷售成本 SUM
3. 銷售毛利（引用度量值）
4. 銷售毛利率 DIVIDE（防除零）
5. 加權平均採購成本（含正確/錯誤做法對比）
6. 零庫存成本修正 LASTNONBLANKVALUE
7. 連續下降天數（時間智能預警）
8. 矩陣總計修正 MAXX + VALUES
9. SWITCH(TRUE) 多條件品類警戒線
10. SUMMARIZE + FILTER 低毛利銷售篩選
11. CONCATENATEX 異常日期追蹤
12. 動態標題 SELECTEDVALUE

#### 10. 資料匯入/匯出功能
- **下載範例 Excel**：一鍵匯出包含 7 張工作表的 `.xlsx` 範例檔
- **匯入格式說明**：彈窗說明每張工作表的欄位需求
- **拖放匯入**：改善拖放計數器避免閃爍
- **匯入回饋**：成功時顯示匯入的工作表名稱列表
- **匯出 JSON**：將當前資料匯出為 JSON 格式

### 技術改進
- 所有圖表統一深色主題配色（軸線/網格/標籤）
- Chart.js datalabels 插件正確註冊
- 移除未使用的 canvas 元素，改用 CSS Treemap
- 修復 drag-and-drop 事件計數器，防止拖離時區域異常消失
- 所有表格新增 hover 高亮效果
