# Gross Profit OS V2.0 - 使用說明書

## 一、系統概述

Gross Profit OS V2.0 是一款專業級毛利率分析互動式資訊看板，以單一 HTML 檔案即可在瀏覽器中運行，無需安裝任何軟體。本系統涵蓋七大分析劇本，支援自訂資料匯入、情境模擬，以及完整的 DAX 公式知識庫。

**系統需求**：現代瀏覽器（Chrome / Edge / Firefox / Safari 最新版本）

**開啟方式**：直接雙擊 `Gross_Profit_OS_V2.0.html` 檔案即可在瀏覽器中開啟。

## 五、功能使用說明

### 5.1 全局分析總覽

首頁展示四大 KPI 指標卡：
- **總營收**：年度銷售收入合計，含 YoY 同比增長
- **總成本**：年度銷售成本合計
- **總毛利**：營收減去成本的絕對值
- **平均毛利率**：毛利 / 營收，低於目標線 35% 會顯示預警

下方包含：
- **瀑布圖**：從營收到毛利的拆解（變動成本 65% + 固定成本 35%）
- **品類獲利分佈**：各產品類別的毛利貢獻甜甜圈圖
- **產品毛利率一覽表**：所有產品的售價/成本/毛利率/銷量/BCG象限
- **預警系統**：自動偵測毛利率是否達標、連續下跌月數、低毛利產品

### 5.2 利潤情境模擬

使用三個滑桿調整：
- **平均單價調整**：-30% ~ +50%
- **銷售總量變動**：-50% ~ +100%
- **單位成本漲跌**：-30% ~ +30%

系統會考慮價格彈性（E=1.5）計算：
- 價格上升 10% => 銷量自動下降 15%（彈性效果）
- 結合手動設定的銷量變動，得出最終預測結果

### 5.3 時間序列趨勢

- **毛利率走勢**：本期 vs 去年同期 + 目標線
- **營收/成本走勢**：月度柱狀對比圖
- **連續下跌偵測**：自動計算最大連續下降月數與對應區間

### 5.4 產品組合 (BCG)

- **BCG 矩陣**：X 軸=相對市占率, Y 軸=市場成長率, 氣泡大小=銷量
- **象限分佈**：明星/金牛/問題兒/瘦狗各有多少產品
- **明細表**：含建議操作（持續投資/穩定收割/評估投入/考慮淘汰）

### 5.5 地域渠道分析

- **地域雷達圖**：各地區在毛利率/配送效率/售價溢價/市場份額/品牌知名度五維度的表現
- **渠道效率圖**：各渠道銷售額/毛利額（柱狀）+ 毛利率（折線）雙軸呈現
- **地區績效表**：五維指標一覽，顏色標識表現好壞

### 5.6 精細成本拆解

- **成本樹狀圖**：按金額佔比大小排列的矩形區塊，直觀呈現成本結構
- **帕累托分析**：成本金額柱狀 + 累積佔比折線，識別 80/20 關鍵成本項
- **成本明細表**：含佔比與累積佔比

### 5.7 客戶價值分析

- **LTV vs 毛利氣泡圖**：X 軸=LTV 總額, Y 軸=毛利貢獻率, 氣泡大小=購買頻率
- **客戶等級分佈**：各等級 LTV 總額佔比
- **客戶明細表**：含等級顏色標識

### 5.8 折扣促銷評估

- **折扣 vs 銷量散點圖**：評估哪些折扣策略帶來最大銷量增幅
- **ROI 對比圖**：銷量增幅/毛利損失比值，判斷促銷活動是否值得
  - ROI >= 5x：優秀（綠色）
  - ROI >= 3x：良好（橘色）
  - ROI < 3x：不佳（紅色）
- **活動明細表**：含 ROI 倍數與評價

### 5.9 DAX 知識庫

收錄 12 條 Power BI 毛利分析專用 DAX 公式，點擊標題即可展開/收合：

1. 銷售收入 SUM - 基礎加總
2. 銷售成本 SUM - 注意零庫存問題
3. 銷售毛利 - 引用度量值確保一致性
4. 銷售毛利率 DIVIDE - 防除零錯誤
5. 加權平均採購成本 - 含正確/錯誤做法對比
6. 零庫存成本修正 LASTNONBLANKVALUE - 向前填充
7. 連續下降天數 - 時間智能預警
8. 矩陣總計修正 MAXX + VALUES - Grand Total Bug 修正
9. SWITCH(TRUE) 多條件品類警戒線
10. SUMMARIZE + FILTER 低毛利銷售篩選
11. CONCATENATEX 異常日期追蹤
12. 動態標題 SELECTEDVALUE

## 七、常見問題

**Q: 圖表顯示空白或異常？**
A: 確認瀏覽器已連接網路（首次載入需下載 Chart.js 與 SheetJS 函式庫）。

**Q: 匯入 Excel 後提示找不到工作表？**
A: 工作表名稱必須完全一致（月度數據、產品數據、成本數據、客戶數據、地區數據、渠道數據、促銷數據）。建議先下載範例 Excel，在此基礎上修改。

**Q: 部分工作表不需要更新？**
A: 可以只包含需要更新的工作表，系統只會覆蓋匯入的部分，其餘保留不變。

**Q: 如何回復到範例資料？**
A: 點擊左側邊欄底部「重置範例資料」按鈕。

**Q: 可以離線使用嗎？**
A: 首次載入後，瀏覽器會快取 CDN 資源。但建議保持網路連線以確保最佳體驗。
